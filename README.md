# Manea (منيع) — Funds Allocation & Expense Tracking System

> *"What if expense fraud was impossible? It already is."*

Manea is a web-based funds allocation and expense tracking system that automates invoice verification using OCR, matches spending against bank transactions, and flags anomalies for auditor review.

---

## 🚀 Overview

Organizations often struggle with manual, fragmented financial processes that leave room for fraud, missing invoices, and untracked spending. Manea solves this by creating a fully automated audit trail — from fund request to invoice verification — across three role-based portals.

---

## ✨ Features

- 📥 **Fund Request Workflow** — Employees submit requests with justification; accountants approve before funds are released
- 🔍 **OCR Invoice Scanning** — Automatically extracts amount, date, and vendor from uploaded invoices
- ⚖️ **Rule-Based Matching Engine** — Validates invoice data against bank transaction records
- 🚨 **Intelligent Flagging** — Auto-detects amount mismatches, overdue submissions, and duplicate invoices
- 👥 **Role-Based Access Control** — Three dedicated dashboards: Employee, Accountant, and Auditor
- 📋 **Full Audit Trail** — Every transaction is traceable back to the employee and approved request
